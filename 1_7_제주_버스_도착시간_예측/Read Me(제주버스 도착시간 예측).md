제주 버스 도착시간 예측 모델링
이 프로젝트는 제주도 버스 운행 데이터를 바탕으로, 특정 정류장에서 다음 정류장까지 소요되는 버스 도착시간을 예측하는 머신러닝 모델을 구축하는 과정입니다.

📋 프로젝트 개요
목표: 과거 버스 운행 로그 데이터를 학습하여, 버스의 다음 정류장 도착시간을 정확하게 예측합니다.

주요 단계:

데이터 전처리: prepare_features 함수를 통한 피처 엔지니어링 수행.

모델 파이프라인 구축: 전처리와 학습 모델을 하나로 묶은 model_pipeline 구성.

성능 평가: 모델의 예측값과 실제 도착시간을 비교하여 모델 정확도 확인.

실전 예측: 새로운 데이터를 모델에 입력하여 예상 도착시간 산출.

🛠 사용된 기술 및 라이브러리
Language: Python

Machine Learning: scikit-learn (파이프라인 구축), xgboost (메인 예측 모델)

Data Handling: pandas, numpy

🚀 주요 모델링 파이프라인
데이터 준비: 원본 데이터를 학습 가능한 형태(수치화)로 변환하는 전처리 함수(prepare_features)를 적용합니다.

학습 및 예측: model_pipeline.predict()를 통해 별도의 복잡한 작업 없이도 데이터 전처리와 예측이 동시에 이루어지도록 설계했습니다.

결과 시각화/출력: 예측된 도착시간을 원본 데이터프레임에 병합하고 소수점 단위로 정리하여 직관적인 결과 리포트를 생성합니다.

💡 활용 및 확장 방안
실시간 적용: 본 모델 파이프라인을 API 서버에 연결하면 실시간 버스 정보 시스템의 '예상 도착시간' 서비스로 활용 가능합니다.

모델 고도화: 현재 적용된 XGBoost 모델 외에 LightGBM 등 다른 앙상블 기법을 파이프라인에 추가하여 성능 개선이 가능합니다.

📝 참고 사항
모델이 정상적으로 예측을 수행하려면 prepare_features 함수가 학습 때와 동일한 로직을 유지하고 있어야 합니다.

예측 결과는 predicted_next_arrive_time 컬럼을 통해 확인할 수 있으며, 필요 시 이를 지도 시각화와 연결하여 사용자에게 제공할 수 있습니다.
